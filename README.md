# myPeripheral-library-STM32



为了减少开发单片机的重复性工作，把平时调过的外设库保存管理起来，方便以后直接调用。

## 主要外设

- OLED显示屏（软件iic）
- SPWM
- PID（增量式，位置式）
- 矩阵键盘
- U8g2 OLED图形库
- SVPWM
